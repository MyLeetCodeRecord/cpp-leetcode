##### lowbit
> 每次lowbit(n)保留最低位的 1
> 
> n的移动不再是n>>1, 而是直接去掉上一个lowbit(n)

```CPP
uint32_t lowbit(uint32_t x) {
    return x & (-x);
}
int hammingWeight(uint32_t n) {
    int cnt = 0;
    while(n!=0){
        n -= lowbit(n);
        cnt++;
    }
    return cnt;
}
```
