#### [Acwing 3400. 统计次数](): `十进制数`

> 好像在期待什么数学推导, 又好像不太期待...
>
> 做题还是要自己分析下数据范围的 😵

```CPP
#include <cstdio>
using namespace std;

int n, k;

int main(){
    scanf("%d %d", &n, &k);
    int ans = 0;
    for(int i=1; i<=n; i++){
        int x = i;
        while(x != 0){
            ans += (x % 10 == k);
            x /= 10;
        }
    }
    printf("%d\n", ans);
    return 0;
}
```
